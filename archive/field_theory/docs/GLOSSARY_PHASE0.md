\# PHASE 0 GLOSSARY (Frozen)



x ∈ \[0, L] : space coordinate.  

t : time coordinate.



ϕ(x,t) : complex scalar field.  

q(x,t), p(x,t) : real components with ϕ = q + i p.  

f(x,t) = |ϕ|, ρ = |ϕ|² : amplitude and density.  

θ(x,t) = arg ϕ : phase.



Primary units (physical scheme):

\- \[x] = L,  \[t] = T,  \[ħ] = L²/T,  \[m] = 1,  \[ϕ] = L^{-1/2},  \[ρ] = L^{-1}.

\- \*\*Coherence coupling\*\*: \[λ] = \*\*L⁴ T⁻²\*\* (so that (λ/4ρ)(∂ₓρ)² has unit L T⁻² in 1D).

\- Target 1D Lagrangian density unit: \[ℒ] = L T⁻².



Natural units (dimensionless mirror runs):

\- ħ = 1, m = 1, identify L ~ T.

\- λ is dimensionless; internal homogeneity holds with \[ϕ] = U^{-1/2}, \[ρ] = U^{-1}.



U(ρ): real potential density, C² near ρ₀.



ρ₀: background density for linearization.



c²(ρ₀) = ρ₀ U′\_ρ(ρ₀) / m : small-amplitude wave speed (primary units).



J = (ħ ρ / m) ∂ₓ θ : 1D current (primary units).



|ϕ|\_ε = √(|ϕ|² + ε²) : node regularization with ε ≪ 1 (default ε = 1.0e−6).
